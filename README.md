# fullstackJS-API
Api para aplicacion sueldoapp

# Requerimientos
* NodeJS instalado
* MongoGB instalado y corriendo

# Pasos para ejecucion
* git clone https://github.com/escuelafullstack/fullstackJS-API.git sueldoapi
* cd sueldoapi
* npm install
* cp .env.example .env
* npm start

# Ejecutar en modo desarrollo
* npm run dev
